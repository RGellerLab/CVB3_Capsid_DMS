This folder contains all the data to perform the phyDMS analysis

1. alignment folder: the pre alignment and the final alignment used for phyDMS
2. phyml_tree folder: the tree used for phyDMS
3. Program output folder: the results of the run
4. updated differential selection: this folder contains the r script to get the sum of absolute differential selection values for each site after removing codons with single mutations 
5. p1_mean_prefs.csv: the average AA preferences for the mutant virus populations used in the phyDMS analysis
